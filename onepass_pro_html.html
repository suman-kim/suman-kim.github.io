시스템 아키텍처 설계 및 개발

확장 가능하고 안정적인 시스템 아키텍처를 설계하여 대규모 출입통제 환경에서도 원활한 서비스를 제공합니다.

  • 얼굴 인식 단말기 연동 모듈 개발
  • 실시간 이벤트 처리 시스템 구축
  • RESTful API 설계 및 구현
  • 마이크로서비스 아키텍처 적용
👥

사용자 관리 시스템

다중 사용자 환경에서 효율적인 권한 관리와 보안성을 보장하는 사용자 관리 시스템을 구축했습니다.

  • 다중 사용자 권한 체계 구현
  • JWT 기반 인증 시스템 구축
  • 사용자 그룹, 직급, 직군 관리 기능 개발
  • 역할 기반 접근 제어(RBAC) 적용
🔐

출입통제 시스템

실시간 출입 이벤트 처리와 유연한 권한 관리를 통해 보안성과 편의성을 동시에 확보한 출입통제 솔루션입니다.

  • 출입 그룹 및 시간 그룹 관리 시스템 개발
  • 실시간 출입 이벤트 처리 및 연동
  • 방문자 관리 및 임시 출입권한 발급 시스템
  • 출입 패턴 분석 및 보고서 생성
⏰

근태관리 시스템

정확한 근태 데이터 수집과 자동화된 보고서 생성을 통해 효율적인 인사관리를 지원하는 시스템을 개발했습니다.

  • 출퇴근 데이터 자동 수집 및 처리
  • 휴가/출장/공휴일 관리 기능 구현
  • Excel 기반 근태 보고서 생성
  • 실시간 근태 현황 대시보드 제공
📱

반응형 웹/모바일 대응

다양한 디바이스에서 최적화된 사용자 경험을 제공하기 위해 반응형 웹 디자인과 모바일 최적화를 구현했습니다.

  • 웹/모바일 분기 처리로 사용자 경험 최적화
  • EJS 템플릿 엔진 활용한 동적 페이지 구성
  • PWA(Progressive Web App) 적용
  • 터치 친화적 UI/UX 디자인
📹

실시간 모니터링

RTSP 기반 실시간 영상 스트리밍과 WebSocket을 통한 실시간 모니터링 시스템으로 보안 관제 기능을 강화했습니다.

  • RTSP 비디오 스트림 처리 (ffmpeg 연동)
  • WebSocket을 이용하여 실시간 영상 전송
  • 다중 카메라 동시 모니터링 지원
  • 실시간 이벤트 알림 및 로깅
